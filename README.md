# 🚀 Desenvolvimento para Servidores I  
📍 Fatec São Roque  

Bem-vindo(a) à disciplina **Desenvolvimento para Servidores I**! 👨‍💻👩‍💻  
Este repositório tem como objetivo central apoiar o aprendizado prático e teórico no desenvolvimento de aplicações para o lado do servidor.

---

## 🎯 Objetivo da Disciplina

Capacitar o aluno a desenvolver aplicações backend utilizando boas práticas de programação, com foco em:

- Programação Orientada a Objetos (POO)
- Integração com banco de dados
- Desenvolvimento de APIs
- Manipulação de dados no formato JSON
- Construção de aplicações web dinâmicas

---

## 🧠 Conteúdos Abordados

### 🔹 Programação Orientada a Objetos com Java
Durante a disciplina, exploramos os principais conceitos da POO utilizando a linguagem Java:

- ✅ Classe e Objeto  
- ✅ Métodos  
- ✅ Encapsulamento  
- ✅ Abstração  
- ✅ Herança  
- ✅ Polimorfismo  
- ✅ Visibilidade (public, private, protected)  
- ✅ Sobrecarga de métodos (Overload)  
- ✅ Sobrescrita de métodos (Override)  

---

### 🔹 Banco de Dados (MySQL)

- Conexão com banco de dados MySQL  
- Execução de comandos SQL  
- CRUD (Create, Read, Update, Delete)  
- Integração entre Java e banco de dados  

---

### 🔹 Fundamentos de PHP

- Sintaxe básica  
- Blocos de código  
- Estruturas condicionais (`if`, `else`, `switch`)  
- Estruturas de repetição (`for`, `while`, `foreach`)  

---

### 🔹 Recursos Avançados

- 📦 Uso de bibliotecas  
- 🌍 Geolocalização  
- 📧 Envio de e-mails  
- 🔗 Integração com APIs externas  

---

### 🔹 APIs e Comunicação de Dados

- Conceito de API  
- API REST (RESTful)  
- Métodos HTTP (GET, POST, PUT, DELETE)  
- Formato de dados JSON  
- Consumo e criação de APIs  

---

## 🛠️ Tecnologias Utilizadas

- Java ☕  
- MySQL 🗄️  
- PHP 🐘  
- JSON 📄  
- APIs REST 🌐
